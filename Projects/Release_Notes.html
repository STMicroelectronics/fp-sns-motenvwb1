Purpose

The FP-SNS-MOTENVWB1 Projects provide applications to aid understanding how to develop an application using the STM32_WPAN Middleware and Motion libraries middlewares. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and supported expansion nucleo boards.

Supported Devices and Boards

  • P-NUCLEO-WB55RG Nucleo-68 board with STM32WB55 MCU supporting Bluetooth™ 5 and 802.15.4 connectivity
  • X-NUCLEO-IKS01A3 Expansion board for four MEMS sensor devices:

  • X-NUCLEO-53L3A2 Multi-targets ranging sensor expansion board based on VL53L3CX for STM32 Nucleo (Optional).
